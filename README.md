# Solar System test app
This a simple react node.js and typescript app example. To run it, do as follows:

### System requirements

node v16.3.0

## Clone repo
```
git clone https://github.com/BellaBe/solar-system.git
```
## Run client
```
cd client 
npm install
npm start
```

## Run server

```
cd server
npm install
npm run start:dev
```
